# SABOREA COLOMBIA - Proyecto Final

##  INFORMACIÓN DEL PROYECTO

**Nombre del Proyecto:** Saborea Colombia

**Descripción:** El proyecto consiste en el desarrollo de una API Backend especializada en la gastronomía colombiana, diseñada para centralizar, organizar y facilitar el acceso a información detallada sobre las diversas regiones culinarias, sus platos típicos, sus ingredientes representativos y los restaurantes locales que mantienen viva esta tradición.

Esta API busca convertirse en la base para aplicaciones móviles, sitios web, emprendimientos gastronómicos o instituciones culturales que deseen promover la culinaria interna, fortalecer la identidad cultural y dar visibilidad a los negocios de comida en todo el país.

Más que una simple base de datos, este proyecto pretende ser un ecosistema digital donde la gastronomía colombiana pueda explorarse de forma estructurada, actualizada y accesible. A través de endpoints bien diseñados, los usuarios podrán:

Consultar regiones culinarias y sus características culturales.

Explorar los platos típicos de cada zona, su historia y composición.

Descubrir restaurantes locales que ofrecen dichos platos.

Integrar esta información en apps de turismo, guías gastronómicas o proyectos educativos.

Además, esta API puede escalar a un modelo de negocio sostenible, permitiendo a restaurantes, guías turísticos y emprendimientos gastronómicos registrarse y promocionar sus servicios, generando oportunidades de visibilidad y crecimiento económico.

El objetivo final es preservar, difundir y modernizar el acceso a la riqueza culinaria de Colombia, conectando la tradición con la tecnología y creando una herramienta capaz de impulsar el turismo cultural tanto nacional como internacional.

**Tecnologías:** NestJS, PostgreSQL, JWT, Swagger, class-validator, class-transformer

**Objetivo:** Desarrollar una API profesional y segura en NestJS que gestione información de regiones culturales, platos típicos y restaurantes, con autenticación de usuarios y operaciones CRUD completas.

**Modalidad:** Trabajo en equipo (5 integrantes)
